# TESA Tour - Система управления туристическими группами

Профессиональная веб-платформа для организации и управления туристическими походами с GPS-трекингом, каналами связи, чатами и экстренными SOS-вызовами.

## Основные возможности

### 1. PWA приложение
- Установка как нативное приложение
- Push-уведомления в реальном времени
- Работа в офлайн режиме
- Быстрая загрузка и производительность

### 2. Система администрирования
- Глобальная панель администратора
- Управление всеми пользователями
- Управление всеми группами
- Отправка системных уведомлений
- Режим администратора для модерации

### 3. Управление группами
- Создание туристических групп
- Система ролей (владелец, администратор, участник)
- Пригласительные ссылки с модерацией
- Черный список пользователей
- GPS-трекинг всех участников

### 4. Каналы
- Официальные каналы для объявлений
- Посты с текстом, фото, видео и файлами
- Предпросмотр медиафайлов
- Реакции на посты (14 эмодзи)
- Закрепление важных сообщений
- Push-уведомления о новых постах

### 5. Групповой чат
- Общение всех участников группы
- Ответы на сообщения
- Реакции под сообщениями
- Обмен файлами и медиа
- Выделение роли (владелец/админ) в чате
- Редактирование своих сообщений
- Удаление сообщений (для админов)

### 6. Опасные зоны
- Отметки опасных участков на карте
- Описание рисков и угроз
- Предупреждения для участников
- Управление зонами администраторами

### 7. SOS-система
- Экстренные вызовы одной кнопкой
- Автоматическое определение координат
- Уведомления через Telegram
- Push-уведомления через PWA
- История всех вызовов

### 8. План-листы маршрутов
- Создание подробных маршрутов
- Контрольные точки на карте
- Отметки о прохождении точек
- Описания локаций

### 9. Система подписки
- Бесплатный тариф "Любитель" (1 группа)
- Платный тариф "Турагентство" (499 руб/мес)
- Неограниченное количество групп для агентств
- Интеграция с ЮКасса для оплаты
- Автоматическое продление подписки

## Технологии

- **Backend**: PHP 7.4+
- **Database**: MySQL 5.7+
- **Frontend**: Vanilla JavaScript, CSS3
- **Maps**: Яндекс.Карты API
- **Notifications**: Telegram Bot API, Web Push API
- **Payment**: ЮКасса API
- **PWA**: Service Workers, Web App Manifest

## Установка

### Требования
- PHP 7.4 или выше
- MySQL 5.7 или выше
- Apache/Nginx с mod_rewrite
- Composer (опционально)

### Шаги установки

1. Клонируйте репозиторий:
```bash
git clone https://github.com/yourusername/tesa-tour.git
cd tesa-tour
```

2. Настройте базу данных:
```sql
CREATE DATABASE tesa_tour CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

3. Импортируйте схему базы данных:
```bash
mysql -u your_user -p tesa_tour < database/schema.sql
```

4. Выполните миграции:
```bash
mysql -u your_user -p tesa_tour < database/migrations/001_add_blacklist_and_approvals.sql
mysql -u your_user -p tesa_tour < database/migrations/002_add_new_features.sql
```

5. Настройте конфигурацию в `config/config.php`:
```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'tesa_tour');
define('DB_USER', 'your_user');
define('DB_PASS', 'your_password');

define('TELEGRAM_BOT_TOKEN', 'your_telegram_bot_token');
define('BASE_URL', 'https://yourdomain.com');
```

6. Настройте права доступа:
```bash
chmod -R 755 uploads/
chmod -R 755 uploads/avatars/
chmod -R 755 uploads/channels/
```

7. Настройте веб-сервер (Apache):
```apache
<VirtualHost *:80>
    ServerName yourdomain.com
    DocumentRoot /path/to/tesa-tour
    
    <Directory /path/to/tesa-tour>
        Options -Indexes +FollowSymLinks
        AllowOverride All
        Require all granted
    </Directory>
    
    ErrorLog ${APACHE_LOG_DIR}/tesa-tour-error.log
    CustomLog ${APACHE_LOG_DIR}/tesa-tour-access.log combined
</VirtualHost>
```

8. Создайте файл `.htaccess` в корне проекта:
```apache
RewriteEngine On
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^(.*)$ index.php [QSA,L]
```

## Настройка интеграций

### Telegram Bot
1. Создайте бота через @BotFather
2. Получите токен бота
3. Добавьте токен в `config/config.php`
4. Установите webhook (опционально)

### ЮКасса
1. Зарегистрируйтесь на yookassa.ru
2. Получите Shop ID и Secret Key
3. Добавьте данные в `SubscriptionController.php`
4. Настройте webhook для уведомлений о платежах

### Яндекс.Карты
1. Получите API ключ на developer.tech.yandex.ru
2. Добавьте ключ в соответствующие представления

## Использование

### Регистрация
1. Перейдите на главную страницу
2. Нажмите "Начать бесплатно"
3. Выберите тип аккаунта (Любитель/Турагентство)
4. Заполните форму регистрации

### Создание группы
1. Войдите в систему
2. Перейдите в раздел "Группы"
3. Нажмите "Создать группу"
4. Заполните информацию о группе

### Приглашение участников
1. Откройте группу
2. Перейдите в "Приглашения"
3. Создайте пригласительную ссылку
4. Отправьте ссылку участникам

### Использование каналов
1. Откройте группу
2. Перейдите в "Каналы"
3. Создайте пост с текстом/медиа
4. Участники получат уведомления

### Отправка SOS
1. Откройте группу
2. Нажмите "Отправить SOS" в меню
3. Добавьте комментарий (опционально)
4. Все участники получат уведомление

## Безопасность

- Пароли хешируются с использованием password_hash()
- Защита от SQL-инъекций через prepared statements
- CSRF защита для форм
- XSS защита через htmlspecialchars()
- Валидация всех входных данных
- Контроль доступа на основе ролей
